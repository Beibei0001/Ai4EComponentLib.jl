# Ai4EComponentLib.jl

Ai4EComponentLib.jl is a Component library based on the [ModelingToolkit](https://mtk.sciml.ai/dev/) acasual modeling system.

## Installation

Import Ai4EComponentLib.jl in the standard way:

```julia
import Pkg; Pkg.add("Ai4EComponentLib")
```
